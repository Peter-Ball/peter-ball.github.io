This mutating welcome sign visualizes a word graph connected by close Levenshtein distances.

The Levenshtein distance is a measure of difference between two strings. It records the number of single-character edits (insertions, deletions, and substitutions) needed to transform one string into another.

Using the starting point Welcome, I recursively grew a graph of words connected by two Levenshtein or less. Each time the page is loaded, the welcome message is mutated by computing an online random walk through the graph.

It's cool to see how close words with very different meanings can be. If you're fast enough, you can get some good tips for your next game of Scrabble.

You can read more on the github page for this site.
